# Saragwali Chokhi (Amla-Version of Dal)
*Saragwali Chokhi*

Prepared using the same method as the dal, but using amla instead of lemon and a simple mustard-seed tempering.

## Ingredients

- dal (type/quantity not specified; 'same method as dal')
- amla (used instead of lemon)
- oil (tempering)
- mustard seeds (tempering)
- ghee (tempering)

## Instructions

1. Cook the dal using the same method as the referenced dal recipe.
2. Use amla instead of lemon.
3. Prepare tempering with oil, mustard seeds, and ghee, and add it to the dal.
